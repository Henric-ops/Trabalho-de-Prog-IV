## 📌 Sobre o Projeto

Aplicação desenvolvida para facilitar o controle acadêmico de TCCs, permitindo gerenciar trabalhos, bancas e arquivos de forma simples e eficiente.

* Interface responsiva
* Upload de arquivos PDF
* CRUD completo
* Feedback visual com notificações
* Design moderno com animações

---

## ✨ Funcionalidades

### 📄 TCCs

* Cadastro, edição e exclusão
* Visualização de arquivos PDF

### 👨‍🏫 Bancas

* Gerenciamento completo das bancas avaliadoras

## ⚙️ Instalação

```bash
git clone <seu-repositorio>
cd crudtcc
composer install
npm install
cp .env.example .env
php artisan key:generate
```

Configure o banco no `.env`:

```env
DB_DATABASE=crudtcc
DB_USERNAME=root
DB_PASSWORD=
```

Execute:

```bash
php artisan migrate
npm run dev
php artisan serve
```
